feat: Add All Matings Ranking and Herd Inventory Management

Major Features:
- All Matings Ranking: New feature to rank all possible matings (cows × sires)
  * Evaluates all cached cow × sire combinations
  * Computes calf EPDs, percentiles, and color bands for each mating
  * Applies gate rules (no trait worse than Gray on gate traits)
  * Scores matings using color-based goodness with configurable weights
  * Displays results grouped by cow with sires ranked best-to-worst
  * Shows all EPD traits with color coding, not just weakness traits
  * Includes sire summary showing how many times each bull ranks #1, #2, etc.
  * Supports filtering by gate-pass and configurable top N per cow (default 5)
  * Top N dropdown options: 1-10 (default 5)
  * Integrated into Mating Calculator with mode toggle (Single/All Matings)

- Herd Inventory Management: New section to view and manage cached animals
  * View all cached animals with name, registration number, sex, and cached date
  * Filter by type (All Animals / Cows Only / Sires Only)
  * Search by name or registration number
  * Select multiple animals via checkboxes for comparison
  * Select All checkbox in header to quickly select filtered results
  * Compare Selected button to load all selected animals into main results
  * View Details button opens modal with individual animal EPD data
  * Delete button to remove animals from cache
  * Auto-refreshes dropdowns after deletion

- Animal Details Modal: Popup window for viewing individual animal details
  * Opens from "View Details" button in inventory
  * Shows animal information (name, registration, sex, sire, dam, birth date)
  * Displays all EPD values with color coding matching main results
  * Close button, click outside, or Escape key to dismiss
  * Scrollable content for long trait lists
